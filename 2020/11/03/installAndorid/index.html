Android Studio 安装

installation
college environment
发布日期:   2020-11-03

Android Studio官网（4.1）

选择安装路径

导入设置：默认勾选 Do not import –> OK

数据分享：默认不发送 –> Don’t send

设置代理：没有 –> Cancel

来到welcome页面 –> Next

安装类型： 默认 Standard –>Next

选择主题： 个人喜欢暗黑 –>Next

等待下载完成 Finish

创建一个新项目 Create New Project

Empty Activity –> Next

配置项目信息 – > Finish

等待出现页面就关掉Android Studio

下载gradle-6.5-bin.zip

下载完成不要解压 放入目录：C:\Users\Administrator.gradle\wrapper\dists\gradle-6.5-bin\6nifqtx7604sqp1q6g8wikw7p

重新打开Android Studio 会下载很多文件，等待下载完成

更改SDK安装路径

首先找到SDK的安装目录，一般默认是在C:\Users\用户名\AppData\Local\Android\sdk文件夹中。然后将SDK文件剪切到任意目录下，然后启动AS，打开 File -> Other Settings -> Default Project Structure，可以看到Android SDK location，将SDK新路径的绝对路径粘贴进去，然后点击Apply即可。
